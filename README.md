## Scheme String Library

A few general-purpose String functions/procedures for Scheme programming language.


### List of Functions

* `str-starts-with?`
* `str-ends-with?`
* `str-exists?`
* `str-find`
* `str-find-all`
* `str-split` alias `str-tokenize`


### Notes

* The underlying algorithms used in these functions are naive/simple.
